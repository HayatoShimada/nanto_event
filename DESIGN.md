# 南砺市イベントページ (NANTS) デザイン仕様書

---

## コンセプト

**「南砺市を好きになるコミュニティーゾーン」**

- **横スクロール主体**: 縦スクロールを廃止し、横方向への移動で時系列や情報の広がりを表現。
- **没入感**: 全画面を活用し、カード型の要素を配置することで、情報の視認性と操作性を高める。
- **動的な体験**: スムーズなスクロールとマイクロインタラクションにより、ユーザーを飽きさせない。

## レイアウト構造

画面全体を「サイドバー」と「メインコンテンツエリア」に分割。

### 1. サイドバー (Left Stick)

画面左側に固定。ナビゲーションと主要アクションを配置。

- **幅**: 250px (PC), アイコンのみ (Mobile)
- **要素**:
  - ロゴ (NANTS)
  - メニュー (HOME, TEAMS, EVENTS, ABOUT, MY PAGE)
  - ユーザープロフィール (簡易表示)
  - 新規作成ボタン (イベント/チーム)

### 2. メインコンテンツエリア (Right Scroll)

画面残りの領域。**横スクロール (Horizontal Scroll)** でコンテンツを表示。

- **構成**: 左側が「最新/未来」、右側へ行くほど「過去」または「詳細」へ。
- **カードレイアウト**:
  - 各コンテンツ（イベント、チーム、記事）をカードとして配置。
  - カードはグリッドあるいはフレックスで並べ、横に流れる。
  - レスポンシブ対応: 画面幅に応じてカードの列数やサイズを調整。

## ページ構成

全てのページは横スクロールのキャンバス上に配置されるイメージ。

### HOME (Dashboard)

- **ヒーローセクション**: 最新の注目イベントを大きく表示。
- **新着イベント**: その次に新しいイベントカードが並ぶ。
- **ニュース/更新情報**: チームからの更新情報。

### EVENTS (Event List)

- **フィルタリング**: カテゴリ、日付でフィルタ。
- **表示**: カレンダービューではなく、横スクロールのタイムラインビュー。
  - 左: 未来のイベント
  - 右: 過去のイベント (アーカイブ)

### TEAMS

- **チーム一覧**: 活動中のチームをカード表示。

### ABOUT

- プロジェクトの理念、南砺市についての紹介。

## 機能要件

### ユーザー (User)

- **属性**: id, role, username, email, padding, address, photoURL
- **認証**: Firebase Auth (Google, Email)

### イベント (Event)

- **属性**: id, name, description, date, location, image, organizer
- **機能**: 作成、編集、参加登録、キャンセル、コメント

### チーム/コラボレーター (Collaborator)

- イベントの共同運営、チームとしての活動。

---

## 実装計画 (Tech Stack)

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS v4 (横スクロールユーティリティ活用)
- **State**: React Context / Hooks
- **Backend**: Firebase (Auth, Firestore, Storage, Functions)
- **Motion**: Framer Motion (横スクロールのアニメーション強化)
